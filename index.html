ShrikLang Docs

What is ShrikLang?

ShrikLang is a cool programming language designed by ANIRUDH SAHU its built with Python, it has variables, functions, loops, errors and moreee!!

Declare a Variable

grab x as 5
say x

User Input

grab name as input
say "Hello " + name

Print Output

say "Hello, World!"

Conditional Logic

grab x as 10
if x > 5 {
  say "x is big!"
} else {
  say "x is small!"
}

Function Definition

craft greet(name) {
  say "Hello " + name
}

greet("Shrik")

Loops

grab i as 0
repeat(i < 5) {
  say i
  grab i as i + 1
}

Raise an Error

oops "Something went wrong!"

How to Run

Save your code in a .shk file and run it using: 

python interpreter.py myfile.shk
